## 📋 使用说明

### 快速开始

1. **安装依赖**
   ```bash
   npm install
   ```

2. **数据库迁移**
   ```bash
   npx prisma migrate dev
   npx prisma generate
   ```

3. **启用 WAL 模式**（备份系统需要）
   ```bash
   node scripts/enable-wal-mode.js
   ```

4. **启动开发服务器**
   ```bash
   npm run dev
   ```

5. **访问系统**
   ```
   http://localhost:3000
   ```
